Micro namespaced localStorage.

IMPORTANT: This is actively under development and the API will be evolving rapidly so you should not use it in production. If you like what you see so far, star it on GitHub so I know you want to use it.

var ls = nsls('example-1');

ls.setItem('what', 'A quick example');
ls.getItem('what')
// "A quick example"

ls.setItem('number', 500);
ls.getItem('number');
//  "500"

ls.removeItem('number');
ls.getItem('number');
// null
